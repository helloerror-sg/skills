# Skills

Development skills I find useful (public). Install into any coding agent via [`npx skills`](https://github.com/vercel-labs/skills).

## Install

```bash
npx skills add helloerror-sg/skills -y -g
```

## Update

```bash
npx skills check -g
npx skills update -g
```
